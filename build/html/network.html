Network Developer Guide¶

This guide is for those developers who wish to see the inner workings of the website and contribute to its design. Below is information about the login system and the SQL queries used on the website. Also there is a very loose style guide for the website and some links to additional helpful information.

Languages Used¶

Valhalla Billiards uses PHP and embedded HTML.

Login System¶

The current login system uses php session variables to determine if someone is logged into the site. After a user inputs their login credintials a simple SQL query is used to check if the account exists.
